# bevy_test_save_3d_scene

Save and load a scene with a 3D mesh. The Mesh and StandardMaterial used are stored using a static HandleId which makes loading the entities easy.

Key: A -> Add Cube
Key: S -> Save Scene
Key: D -> Clear Scene
Key: L -> Load Scene
